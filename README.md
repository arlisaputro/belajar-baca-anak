# 🎓 Belajar Membaca - Anak TK

Web interaktif belajar membaca Bahasa Indonesia untuk anak TK. Dibuat dengan HTML, CSS, dan JavaScript murni — tanpa framework, tinggal buka di browser!

## 🌟 Fitur

- **🔤 Belajar Huruf A-Z** — Kartu huruf berwarna-warni dengan suara pengucapan
- **🗣️ Belajar Suku Kata** — Pilih huruf konsonan, pelajari:
  - Suku kata dasar (ba, bi, bu, be, bo)
  - Kombinasi vokal + suku kata (a-ba, i-ba, u-ba, ...)
  - Kombinasi suku kata + vokal (ba-a, ba-i, ba-u, ...)
  - Kombinasi suku kata + suku kata (ba-ba, ba-bi, ba-bu, ...)
- **📖 Belajar Kata** — Kata-kata sederhana dikelompokkan per huruf A-Z, lengkap dengan emoji dan pemisahan suku kata
- **🎯 Quiz** — Tebak kata dari gambar emoji, ada skor dan feedback suara

## 🚀 Demo

## 💻 Cara Pakai

Cukup buka file `index.html` di browser, tidak perlu install apapun.

```bash
git clone https://github.com/arlisaputro/belajar-baca-anak.git
cd belajar-baca-anak
open index.html
```

## 🔊 Text-to-Speech

Menggunakan Web Speech API bawaan browser untuk pengucapan Bahasa Indonesia. Pastikan browser mendukung fitur ini (Chrome, Edge, Safari).
